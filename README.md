# Mercato Ecommerce App

A static ecommerce storefront built with HTML, CSS, and JavaScript.

## Features

- Responsive product catalog
- Search and category filtering
- Add-to-cart drawer
- Quantity controls
- Checkout summary
- Cart persistence with local storage
- Netlify-compatible static deployment

## Run locally

Open `index.html` in a browser.

## Deploy to Netlify

Use these Netlify settings:

- Build command: leave empty
- Publish directory: `.`

The included `netlify.toml` already sets the publish directory.
